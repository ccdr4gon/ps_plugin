# Photoshop 绘画键位脚本 (AutoHotkey v2)

把 Photoshop 改成绘画友好的键位。**仅在 Photoshop 前台时生效**，其他程序完全不受影响。

---

## ⚙️ 首次使用：必做的 3 步

### 第 1 步：装 AutoHotkey v2 并运行脚本
1. 到 [autohotkey.com](https://www.autohotkey.com/) 下载安装 **AutoHotkey v2.0**（注意是 **v2**，不是 v1，语法不同）。
2. 双击 `ps-paint-keys.ahk` 运行。屏幕右下角出现托盘提示「PS 键位已加载」即成功。
3. （可选，开机自启）按 `Win+R` 输入 `shell:startup` 回车，把 `ps-paint-keys.ahk` 的快捷方式拖进去。

### 第 2 步：在 PS 给「涂抹工具」配快捷键 K
涂抹/模糊/锐化这组工具默认都没有快捷键，脚本的 `C` 键转发的是 `K`，所以要先在 PS 里给**涂抹工具**配上 K（我们用涂抹作为默认的模糊笔刷）：
1. PS 菜单 **编辑 → 键盘快捷键**（快捷键：`Ctrl+Alt+Shift+K`）。
2. 顶部「**快捷键用于**」下拉选 **工具 (Tools)**。
3. 在列表里滚动找到 **涂抹工具 / Smudge Tool**（在 模糊/锐化 一组里）。
4. 点它右侧空白框，按一下 **K** 键。
5. 右上角点 ✓（接受）→ 再点确定保存。

> 为什么是 K：它是 PS 官方推荐的空闲键，不会和其他快捷键冲突。

### 第 3 步：在 PS 录一个「水平翻转视图」动作绑 F2
脚本的 `T` 键转发 `F2`，需要让 F2 = 水平翻转视图：
1. PS 菜单 **窗口 → 动作**（打开动作面板）。
2. 点面板底部「新建动作」图标 → 名称随意 → **功能键**下拉选 **F2** → 点「记录」。
3. 点 PS 菜单 **视图 → 水平翻转**（非破坏性，只翻显示）。
4. 点动作面板的「停止」方块。
   完成后 F2 = 翻转视图，脚本的 T 即生效。

> 为什么不直接让脚本点菜单：中文界面菜单字母对不上、还会触发 Alt 菜单卡顿，用动作绑 F2 最稳。

### 前置确认
- PS 进程名为 `Photoshop.exe`（一般都是；若不是，改脚本里 `ahk_exe Photoshop.exe`）。
- `A` 键的旋转视图需 PS 已开 **首选项 → 性能 → 使用图形处理器**（能用旋转视图就说明已开）。

## ⌨️ 快捷键一览

| 键 | 功能 | 实际发送 / 说明 |
|---|---|---|
| `Q` | 画笔 | B |
| `E` | 橡皮 | E |
| `R` | 自由套索 | L |
| `A` | 旋转视图工具 | R（切过去后鼠标拖着转，转完按 Q 回画笔） |
| `V` | 魔棒 | W |
| `Z` | 直线 / 形状工具 | U |
| `C` | 涂抹工具（默认模糊笔刷） | K（需第 2 步配置） |
| `W` | 放大 | Ctrl + |
| `S` | 缩小 | Ctrl − |
| `T` | 水平翻转视图 | F2（需第 3 步配置） |
| `Ctrl+F` | 前景色填充选区 | Alt+Delete（无选区则填整层） |
| `Ctrl+Alt+左键拖` | 改笔刷大小 | 横向拖=大小、纵向拖=硬度/透明度（转 PS 原生 Alt+右键拖） |
| `Ctrl+左键拖` | **已禁用**（防误触移动/复制） | 拖动被吞掉、内容不动；移动改用 `Ctrl+T`。`Ctrl+左键单击`照常保留 |
| `F1` | **暂停 / 恢复脚本** | 见下方注意事项 |

---

## ⚠️ 注意事项

- **打字前先按 F1 暂停**：`Q E R A V Z C W S T` 是裸字母键，脚本启用时这些字母会被改成工具/缩放。用**文字工具**打字、或在任何输入框输入含这些字母的内容前，先按 `F1` 暂停；打完再按 `F1` 恢复。
- **Ctrl 组合键不受影响**：`Ctrl+S`（保存）、`Ctrl+A`（全选）等照常工作，脚本只拦截裸字母键。
- **Ctrl+左键「拖」已禁用**：PS 默认 Ctrl+拖会移动/复制内容，画画时极易误触；脚本只吞「拖动」，`Ctrl+左键单击`（如点图层缩览图载入选区）仍正常。要移动内容请用 `Ctrl+T` 自由变换。
- **改笔刷要在画笔/橡皮等工具下用**：移动工具下 `Ctrl+Alt+拖` 是 PS 的复制图层，脚本不在那场景接管。
- **配置丢失**：第 2 步建议在快捷键对话框存成一套新的快捷键组；动作（第 3 步）会随 PS 保存。

---

## 改键位 / 维护

- 所有映射都在 `ps-paint-keys.ahk` 里，每行注释写明了对应 PS 的哪个键，照着改即可。
- 改完脚本后**重启脚本**：托盘图标右键 → **Reload Script**（不是 PS 的 Reload）。
- 临时全关：托盘图标右键 → Exit；或按 F1 暂停。
